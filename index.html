源码安装指南



依赖

  • Python 3
  • PyQt4 (python3-pyqt, python3-qt4)
  • （可选）Python-notify2 (python3-notify2)
  • (Debian 用户) pyqt4-dev-tools qt4-linguistic-tools

    • 注意：不同的发行版可能使用不同的软件包名称，或者在软件库中找不到python-notify2。我们将会创建这些依赖的.rpm以及.deb包，在此之前，请用 PIP 安装。


    安装步骤

    1. 安装WeCase所需要的依赖。

    2. 解压下载好的压缩包或者用 git clone 

    git clone --recursive git://github.com/WeCase/WeCase.git

    3. 安装 SDK

    cd WeCase/sdk
    sudo python3 setup.py install
    cd ..

    4. 安装 WeCase

    mkdir build
    cd build
    ../configure
    make
    sudo make install

    5. 运行 WeCase

    wecase
